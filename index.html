Conectar Google Sheets

Los datos se van a guardar en una hoja de cálculo compartida. Seguí estos pasos una sola vez:

1. Creá el Google Sheet Abrí sheets.new, nombralo "Gastos Familiares" y copiá el ID de la URL:
https://docs.google.com/spreadsheets/d/[ESTE ID]/edit
2. Creá el Apps Script En el Sheet: Extensiones → Apps Script. Borrá el código existente y pegá el script que está más abajo. Hacé clic en Implementar → Nueva implementación → Aplicación web, poné acceso para "Cualquiera" y copiá la URL del deployment.
